# sphsol-release

Here we provide a release version of the sphsol code used in the manuscript "Giant Craters on 253 Mathilde Revealing the Cohesive Porous Interior of Carbonaceous Parent Bodies". A documentation of the code is available at https://sphsol-tutorial.readthedocs.io.
